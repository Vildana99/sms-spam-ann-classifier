# SMS Spam Classification using ANN

This project classifies SMS messages into spam and ham categories using Natural Language Processing (NLP) and an Artificial Neural Network (ANN).

## Dataset
SMS Spam Collection Dataset:
https://archive.ics.uci.edu/dataset/228/sms+spam+collection

## Technologies
- Python
- TensorFlow / Keras
- Pandas
- Scikit-learn
- Matplotlib

## Model Architecture
- Tokenizer
- Padding
- Embedding
- GlobalAveragePooling1D
- Dense (ReLU)
- Dense (Sigmoid)

## Results
Test accuracy: ~98%
